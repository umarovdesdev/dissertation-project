# figures/ — thesis & defense figure generation

Standalone workspace for generating the figures used in the dissertation and
defense. **Not part of the demo build** — moved here out of `demo/public/` so it
is no longer bundled/served by the React app. Intermediate by nature: the
scripts may go stale as data and models change.

- `scripts/` — figure-generation Python scripts (see per-script table below)
- `figures_mine/` — generated PNG/CSV/TXT outputs
- `scripts/README.md` — detailed inputs, run order, and the `predictions.npz` recipe for fig7

Scripts read inputs from `demo/public/datasets/`, `experiments/src/`, and
`E:/datasets/`; outputs land in `figures_mine/`.

## Scripts (short)

| Script | Produces |
|---|---|
| `fig1_grid_per_class.py` | Grid of sample fundus images per DR grade (0–4) across datasets |
| `fig2_lesion_overlays.py` | IDRiD lesion segmentation masks overlaid on fundus images |
| `fig3_dataset_contents.py` | Dataset composition chart + class-distribution CSV |
| `fig4_flowchart.py` | Preprocessing/pipeline flowchart |
| `fig5_architecture.py` | Model architecture diagram (artistic) |
| `fig6_model_graph.py` | Model computation graph (Graphviz, ONNX fallback) + layer summary |
| `fig7_pr_curves.py` | Per-class precision–recall curves (needs `figures_mine/data/predictions.npz`) |
| `fig9_confusion_matrix.py` | Side-by-side confusion matrices, Config C vs Config D |
| `copy_ready_figures.py` | Copies ready training-curve (fig8) and confusion-matrix (fig9) PNGs from `demo/public/images/results/` |
| `prepare_local.py` | Builds local loss/F1 comparison charts for the configs |
| `requirements.txt` | Python dependencies for the above |

fig8 (training curves) and fig10 (web-app screenshots) are not generated by a
dedicated script — see `scripts/README.md`.
