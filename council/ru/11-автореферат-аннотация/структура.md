# Шаблон: Автореферат / Аннотация (трёхъязычная)

> **Назначение:** краткое изложение диссертации, публикуется на сайте до защиты и
> рассылается. В диссовете МУИТ готовится **на трёх языках** (казахский, русский, английский),
> объёмом **до 15 страниц** на язык (§6.9 Положения).
> **Это самый частозапрашиваемый документ — следуйте этой структуре, а не общим шаблонам.**

## Обязательные требования

- Языки: **қазақша / русский / English** — три отдельные версии, одинаковые по структуре.
- Объём — до 15 страниц на каждом языке.
- Содержит: тему, цель, задачи, методы, основные положения на защиту, новизну, значимость.
- Оформление — по [[../02-оформление/оформление-по-госту|ГОСТ]] (титул + оборот для автореферата-брошюры).

## Структура (порядок разделов «Общей характеристики работы»)

1. **Актуальность темы исследования** — состояние проблемы, обоснование необходимости.
2. **Цель исследования** — одна формулировка.
3. **Задачи исследования** — нумерованный список (обычно соответствуют главам).
4. **Объект исследования.**
5. **Предмет исследования.**
6. **Методология и методы исследования.**
7. **Научная новизна** — нумерованный список новых результатов.
8. **Положения, выносимые на защиту** — нумерованный список (каждое защищаемое положение).
9. **Теоретическая значимость.**
10. **Практическая значимость** (включая внедрение, акты внедрения).
11. **Достоверность результатов** — чем обеспечена (объём данных, метрики, сравнение).
12. **Эмпирическая / экспериментальная основа** — датасеты, объём данных.
13. **Апробация результатов** — конференции, семинары; **связь с научными программами**
    (подп. 2 п. 3 ст. 20 Закона РК «О науке…», госпрограммы).
14. **Публикации** — со ссылкой на [[../18-список-научных-трудов/структура|список трудов]]
    (WoS/Scopus с квартилем и процентилем; ККСОН; конференции; авторские свидетельства).
15. **Структура и объём** диссертации.
16. **Основное содержание работы** — краткое изложение по главам.
17. **Заключение** — основные выводы.
18. **Список опубликованных работ по теме диссертации.**

```
ОБЩАЯ ХАРАКТЕРИСТИКА РАБОТЫ
Актуальность темы исследования: ...
Цель исследования: ...
Задачи исследования:
   1) ...
   2) ...
Объект исследования: ...
Предмет исследования: ...
Методология и методы: ...
Научная новизна:
   1) ...
Положения, выносимые на защиту:
   1) ...
Теоретическая значимость: ...
Практическая значимость: ...  (акты внедрения: ...)
Достоверность результатов: ...
Эмпирическая основа: ... (датасеты, объём)
Апробация / связь с программами: ...
Публикации: всего N, в т.ч. WoS/Scopus — n, ККСОН — n, конференции — n, свид-ва — n
Структура и объём: введение, N глав, заключение, ... стр.

ОСНОВНОЕ СОДЕРЖАНИЕ РАБОТЫ
   Во введении ...
   В первой главе ...
   Во второй главе ...
   В третьей главе ...

ЗАКЛЮЧЕНИЕ
   1) ...

СПИСОК ОПУБЛИКОВАННЫХ РАБОТ ПО ТЕМЕ ДИССЕРТАЦИИ
```
