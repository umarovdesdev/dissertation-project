# Шаблон: Явочный лист членов диссертационного совета (Приложение 4)

> **Назначение:** официально утверждённая форма (Приложение 4 Положения). Подтверждает явку
> и кворум на заседании по защите (см. [[../01-процесс-защиты/регламент-заседания]]).
> **Подписывает:** Учёный секретарь.

## Шапка

```
ЯВОЧНЫЙ ЛИСТ
членов диссертационного совета по направлению 8D061 — Информационно-коммуникационные технологии
Дата заседания: «__» ______ 20__ г.   Протокол № ___
Докторант: <ФИО>
Шифр и наименование образовательной программы: <8D061XX — ...>
```

## Форма таблицы

| ФИО членов совета | Учёная степень / звание | Явка на заседание (присутствовал / отсутствовал) |
|-------------------|--------------------------|---------------------------------------------------|
| | | |

> Включаются как постоянные, так и временные члены, привлечённые под данную защиту
> (см. [[../00-нормативная-база/состав-диссовета]]).

## Подпись

```
Учёный секретарь диссертационного совета _______________ /ФИО/
```
