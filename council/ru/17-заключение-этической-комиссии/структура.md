# Шаблон: Заключение этической комиссии (Приложение 2)

> **Назначение:** официально утверждённая форма (Приложение 2 Положения о диссовете).
> Готовится на этапе [[../01-процесс-защиты/предварительная-экспертиза|предварительной экспертизы]].
> **Подписывают:** Председатель + Секретарь этической комиссии.

## Форма (таблица из 9 строк)

| № | Показатель | Содержание |
|---|-----------|------------|
| 1 | ФИО докторанта | |
| 2 | Специальность (образовательная программа) | 8D061XX — ... |
| 3 | Период обучения | 20__–20__ |
| 4 | Тема и дата её утверждения | «...», утв. «__» ______ 20__ г. |
| 5 | Данные о научных консультантах | отечественный / иностранный (ФИО, степень, организация) |
| 6 | Объекты исследования | (данные / люди / организации и т. п.) |
| 7 | Нарушения в планировании / оценке / отборе / проведении исследования | выявлены / **не выявлены** |
| 8 | Нарушения в распространении результатов | выявлены / **не выявлены** |
| 9 | Защита прав, безопасности и благополучия объектов исследования | обеспечена / не обеспечена |

## Подписи

```
Председатель этической комиссии _______________ /ФИО/
Секретарь этической комиссии    _______________ /ФИО/
«__» ______ 20__ г.
```
