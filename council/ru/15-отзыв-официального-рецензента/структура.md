# Шаблон: Письменный отзыв официального рецензента (Приложение 3)

> **Назначение:** официально утверждённая форма отзыва рецензента (Приложение 3 Положения о
> диссовете). **Строгая табличная форма** — следовать ей точно.
> Оглашается на защите (см. [[../01-процесс-защиты/регламент-заседания]], п. 6).

## Шапка

```
Письменный отзыв официального рецензента
на диссертационную работу <ФИО докторанта>
на тему «<...>» на соискание степени доктора философии (PhD)
Образовательная программа: <8D061XX — наименование>
```

## Форма таблицы

| № | Критерии | Соответствие критериям (подчеркнуть нужное) | Обоснование позиции (замечания — курсивом) |
|---|----------|---------------------------------------------|---------------------------------------------|
| | | соответствует / частично / не соответствует | |

## 13 критериев оценки

1. **Тема соответствует** направлениям развития науки / государственным программам.
2. **Важность для науки** (актуальность, значимость темы).
3. **Принцип самостоятельности** — работа выполнена соискателем самостоятельно.
4. **Принцип внутреннего единства** (подпункты 4.1–4.5): целостность работы, связь глав,
   единая логика, соответствие выводов задачам, отсутствие противоречий.
5. **Принцип научной новизны** (подпункты 5.1–5.3): новизна постановки, методов, результатов.
6. **Обоснованность основных выводов** — выводы вытекают из результатов.
7. **Основные положения, выносимые на защиту** (7.1–7.5: оценка по каждому положению отдельно).
8. **Принцип достоверности** (8.1–8.5): корректность методов, данных, экспериментов,
   статистической обработки, воспроизводимость.
9. **Принцип практической ценности** (9.1–9.3): применимость, внедрение, экономический эффект.
10. **Качество написания и оформления** (соответствие [[../02-оформление/оформление-по-госту|ГОСТ]]).
11. **Замечания** (перечень критических замечаний — курсивом).
12. **Научный уровень статей** (соответствие требованиям к публикациям — WoS/Scopus, ККСОН;
    см. [[../00-нормативная-база/требования-к-публикациям]]).
13. **Решение официального рецензента** — итоговый вывод о возможности присуждения степени
    доктора философии (PhD) по образовательной программе <...>.

## Реквизиты подписанта

```
Официальный рецензент:
   ФИО: ...
   Учёная степень / звание: ...
   Должность, организация: ...
   Подпись: ___________  Дата: «__» ______ 20__ г.
```

> Замечания по тексту критериев оформляются **курсивом** в столбце «Обоснование позиции».
> В столбце «Соответствие критериям» нужный вариант **подчёркивается**.
