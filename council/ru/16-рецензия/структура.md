# Шаблон: Рецензия / неофициальный отзыв

> **Назначение:** рецензия на диссертацию/автореферат или неофициальный отзыв, поступивший
> на опубликованную аннотацию. Свободная форма; на защите оглашаются неофициальные отзывы
> (см. [[../01-процесс-защиты/регламент-заседания]], п. 8).
> Отличать от строго формализованного [[../15-отзыв-официального-рецензента/структура|отзыва официального рецензента]].

## Шапка

```
РЕЦЕНЗИЯ (ОТЗЫВ)
на диссертационную работу (автореферат) <ФИО докторанта>
на тему «<...>», представленную на соискание степени доктора философии (PhD)
Образовательная программа: <8D061XX>
```

## Структура содержания

1. **Актуальность темы.**
2. **Краткое содержание работы** — структура, основные результаты.
3. **Научная новизна и достоверность.**
4. **Практическая значимость.**
5. **Замечания** (при наличии).
6. **Вывод** — соответствие требованиям к работам на соискание PhD; рекомендация.

## Реквизиты подписанта

```
   ФИО: ...
   Учёная степень / звание, должность, организация: ...
   Подпись: ___________  Дата: «__» ______ 20__ г.
```
