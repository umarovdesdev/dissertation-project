## 2. Слайд мазмұны

**7 эксперимент — 6 гипотеза — 8 датасет**

| Exp | Гипотеза | Дизайн | Оқыту → Тест | Табыстылық критерийі |
|-----|----------|--------|--------------|---------------------|
| 1 | H-1: Preprocessing Dominance | 2×2 факторлық (A/B/C/D) + Config N | EyePACS (5-fold CV) | EH-3: ΔF1 ≥ 5pp ∧ ΔAUC ≥ 0.02 ∧ Δκ ≥ 0 |
| 2 | H-2: Component Ablation | 7 деңгейлі ablation + CLAHE/σ sweep | EyePACS (5-fold CV) | Per-class F1 оптимум, CNR/VVI жақсаруы |
| 3 | H-4: Transferability | Zero-shot transfer (қайта оқытусыз) | EyePACS → APTOS 2019 | G = F1_ext / F1_EyePACS ≥ 0.85 |
| 4 | H-5: Explainability | Grad-CAM + ALO/IoU | IDRiD (50 img) + Clinical (50 img) | ALO_V5 > ALO_baseline |
| 5 | H-7: Degradation Resistance | Δ = F1_val − F1_ext салыстыру | EyePACS → IDRiD, Messidor-2 | Δ_V5 < Δ_baseline |
| 6 | H-6: Device Domain Shift | Cross-camera zero-shot | EyePACS → DDR, ODIR-5K, RFMiD | F1 stable: Canon, Topcon, Kowa, Zeiss |
| 7 | — | Small data training | IDRiD (5-fold CV) → Clinical (held-out) | Weighted F1, κ on 60 images |

**Статистикалық валидация:**

| Тест | Мақсаты | Экспт |
|------|---------|-------|
| McNemar | Жұптық жіктеу салыстыру (A↔B, C↔D) | 1 |
| DeLong | ROC-AUC айырмашылығының маңыздылығы | 1, 3 |
| Bootstrap CI (≥ 1000) | 95% сенімділік интервалдары | 1–7 |
| Bonferroni/Holm | Көп салыстыру түзетуі | 1, 2 |
| Mixed-effects model | Fold — кездейсоқ эффект ретінде | 1 |

**Эксперименттік валидтілік (threats to validity):**

| Валидтілік қаупі | Бақылау шарасы |
|-----------------|---------------|
| Data leakage | Patient-level CV: бір пациенттің барлық суреттері бір fold-та |
| Overfitting | Early stopping (patience=5) + train-test gap ≤ 15pp |
| Нормализация confound | Config N: baseline + dataset-specific norm (pipeline-сыз) |
| Қайта өндірілмеушілік | seed=42, deterministic=true, тұрақты аугментация |

## 3. Баяндаушы сөзі

Зерттеудің эксперименттік бағдарламасы 7 эксперименттен тұрады және 6 гипотезаны 8 датасетте тексереді. Эксперименттер логикалық тізбек бойынша құрылған — ішкі валидациядан сыртқы валидацияға, сандық бағалаудан клиникалық түсіндірмелілікке дейін.

**Бірінші эксперимент** — зерттеудің негізі. 2 мал 2 факторлық дизайн арқылы preprocessing-тің classification-ға тигізетін әсерін CNN архитектурасынан бөліп көрсетеді. Төрт конфигурация: A, B, C, D. Preprocessing доминанттылығы тек доминанттылық критерийі — үш шарт бір мезгілде — екі архитектурада да бөлек тексеріледі. Сонымен қатар Config N — baseline + dataset-specific нормализация — қосылған. Бұл конфигурация маңызды confound-ты шешеді: baseline ImageNet нормализация қолданады, ал V5 pipeline dataset-specific нормализация қолданады. Егер Config N baseline-ға жақын нәтиже берсе — демек жақсару нормализациядан емес, preprocessing pipeline-нан.

**Екінші эксперимент** — ablation. Pipeline-ның қай компоненттері нәтижеге ең көп ықпал етеді? 7 деңгей — baseline-нан бастап толық V5-ке дейін, бір-бірлеп stage қосылады. Қосымша — CLAHE параметрлері мен flat-field sigma бойынша sweep.

**Үшінші эксперимент** — тасымалдаушылық. EyePACS-те оқытылған модельді APTOS 2019-да zero-shot бағалау. Табыстылық критерийі: generalization ratio 85 пайыздан жоғары. Модельді қайта оқытпаймыз — тек pipeline мен CNN-ді сол қалпында тестілейміз.

**Төртінші эксперимент** — түсіндірмелілік. Grad-CAM арқылы модельдің назар аймағын визуализациялап, ALO метрикасымен IDRiD-тың зақым маскаларымен сандық салыстыру. Preprocessing модельдің назарын шынымен зақым аймағына бағыттайды ма — осы сұраққа жауап.

**Бесінші эксперимент** — клиникалық деградация. EyePACS-те оқытылған модель IDRiD пен Messidor-2-де қаншалықты нашарлайды? Preprocessing бұл нашарлауды азайтады ма?

**Алтыншы эксперимент** — құрылғы domain shift. Төрт камера өндірушісінде — Canon, Topcon, Kowa, Zeiss — модель тұрақтылығын тексереді.

**Жетінші эксперимент** — шағын деректерде оқыту. IDRiD-тың 516 суретінде 5-fold CV, нәтижесін 60 клиникалық суретте бағалау.

Статистикалық валидация тұрғысынан: McNemar тесті жұптық жіктеу нәтижелерін салыстырады, DeLong тесті ROC-AUC айырмашылығын тексереді, bootstrap 95 пайыздық сенімділік интервалдарын есептейді. Салыстыру кезінде Bonferroni/Holm түзетуі қолданылады — бұл жалған оң нәтиженің ықтималдығын бақылауда ұстайды.

Эксперименттік валидтілік — төрт деңгейде қамтамасыз етіледі. Patient-level кросс-валидация data leakage-ді болдырмайды. Early stopping мен train-test gap бақылауы overfitting-ті анықтайды. Config N нормализация confound-ын шешеді. Тұрақты seed пен deterministic режим — нәтижелердің қайта өндірілуін қамтамасыз етеді.

## 4. Қосымша

- **2×2 факторлық дизайн** — екі тәуелсіз фактор (preprocessing деңгейі × CNN архитектурасы) бір мезгілде тексеріледі. Main effect (әр фактордың жеке әсері) мен interaction effect (факторлардың бірлескен әсері) бөлуге мүмкіндік береді. 4 конфигурация: A (baseline+ResNet), B (V5+ResNet), C (baseline+EfficientNet), D (V5+EfficientNet).
- **Config N (нормализация контролі)** — baseline preprocessing + dataset-specific нормализация (V5 pipeline stages жоқ). Мақсаты: baseline ImageNet нормализация, ал V5 dataset-specific нормализация қолданатындықтан, нәтижедегі жақсару pipeline-нан ба, әлде нормализация статистикасының өзгеруінен бе — осыны ажырату. Егер N ≈ A/C болса → нормализация жеткіліксіз; егер A/C < N < B/D болса → екеуі де үлес қосады.
- **Ablation (7 деңгей)** — кезең-кезеңмен pipeline компоненттерін қосу: Level 0 (baseline) → Level 1 (+flip) → Level 2 (+rotation) → Level 3 (+isotropic+mask) → Level 4 (+flat-field) → Level 5 (+CLAHE) → Level 6 (full V5). Әр деңгейдегі F1 өсімі — сол кезеңнің маржиналды үлесі.
- **CLAHE sweep** — clip_factor мен global_threshold параметрлерінің мәндерін жүйелі түрде өзгертіп, DR 1 мен DR 2 per-class F1-ге әсерін анықтау. Мақсаты: оптималды параметр аймағын табу.
- **Flat-field σ sweep** — σ = 0.05·D-ден 0.10·D-ге дейін өзгертіп, жарықтандыру түзетуінің сезімталдығын бағалау. D = FOV диаметрі пиксельде.
- **Zero-shot transfer** — модельді сыртқы датасетте ешқандай қайта оқытусыз (fine-tuning жасамай) бағалау. Generalization ratio G = F1_external / F1_EyePACS.
- **ALO (Attention–Lesion Overlap)** — ALO = |GradCAM ∩ lesion| / |lesion|. Зақым масктарының CNN назар аймағымен қабаттасуын зақым жағынан бағалайды. Жоғары ALO = модель зақым аймағына назар аударады.
- **IoU (Intersection-over-Union)** — IoU = |GradCAM ∩ lesion| / |GradCAM ∪ lesion|. Симметриялық кеңістіктік дәлдік — ALO-дан айырмашылығы, Grad-CAM аймағының артық кеңейгенін де ескереді. Қосымша метрика.
- **Degradation Δ** — Δ = F1_EyePACS_val − F1_external. Сыртқы датасетте қаншалықты нашарлау бар екенін көрсетеді. Δ_V5 < Δ_baseline → preprocessing нашарлауды азайтады.
- **McNemar тесті** — 2×2 contingency table негізінде екі модельдің жұптық жіктеу нәтижелерін салыстырады. Бір модель дұрыс, екіншісі қате жіктеген жағдайлардағы айырмашылықты тексереді. H₀: екі модельдің қате бөлуі бірдей.
- **DeLong тесті** — екі ROC-AUC арасындағы айырмашылықтың статистикалық маңыздылығын тексереді. Корреляцияланған ROC қисықтарын ескереді (бір тест жиынтығында).
- **Bootstrap CI** — бастапқы тест жиынтығынан кемінде 1000 рет кездейсоқ іріктеу жасап, метрика мәндерінің 95% сенімділік интервалын есептеу. Шағын датасеттер (IDRiD=516, Clinical=60) үшін міндетті.
- **Bonferroni/Holm түзетуі** — көп салыстыру кезінде α шегін түзетеді. Бір эксперимент ішінде бірнеше салыстыру жасалғанда жалған оң нәтиже (Type I error) ықтималдығын бақылауда ұстайды.
- **Mixed-effects model** — fold-ты кездейсоқ эффект ретінде модельдейді. Fold арасындағы вариацияны ескере отырып, preprocessing-тің тұрақты әсерін бағалайды.
- **Train-test gap** — оқыту F1 мен тест F1 арасындағы айырмашылық (пайыздық пунктпен). 15pp шегі — overfitting индикаторы: gap > 15pp → модель оқыту деректеріне артық бейімделген.
- **Data leakage** — оқыту деректерінен тест деректеріне ақпараттың ағуы. Patient-level CV арқылы алдын алу: бір пациенттің сол мен оң көз суреттері бір fold-та, ешқашан бөлінбейді.
