## 2. Слайд мазмұны

**Қолданыстағы тәсілдер**
- Архитектураға бағытталған: ensemble, гибрид CNN+ViT (Vision Transformer), SE-attention блоктары
- Препроцессинг (preprocessing): CLAHE (Contrast-Limited Adaptive Histogram Equalization), нормализация — модельден тыс саты ретінде
- Cross-dataset: domain shift архитектура деңгейінде шешілмеген
- Түсіндірілгіштік (explainability): Grad-CAM визуалды heatmap, сандық метрика жоқ

**Теориялық негіздеме — model = preprocessing + CNN**

Қолданыстағы тәсіл: data = prep(I) → ŷ = CNN(data)
Ұсынылған тәсіл: ŷ = CNN_θ(P(I, s))

**Себептік тізбек:**
V5 препроцессинг → микроваскулярлы белгілердің көрінуі → CNN feature extraction тұрақтылығы → DR классификациясының жақсаруы

## 3. Баяндаушы сөзі

Қолданыстағы зерттеулерді төрт бағытта талдадық: 
- архитектураға бағытталған тәсілдер, 
- препроцессинг тәжірибесі, 
- cross-dataset мәселесі 
- және түсіндірілгіштік (). 

Қолданыстағы зерттеулердің басым бөлігі дәлдікті модельдің архитектурасын күрделендіру арқылы арттырады, ал препроцессингтің CNN feature space-іне әсері жүйелі зерттелмеген. 

Зерттеудің теориялық негізі — препроцессинг CNN модельдің ажырамас компоненті. 

Себептік тізбек арқылы препроцессинг микроваскулярлы зақымдардың көрінуін арттырады, бұл CNN feature extraction-ды тұрақтандырып, классификацияны жақсартады. 

Тұжырымды бекіту үшін төрт факторлық дизайн құрылды — main effect пен interaction effect бөлуге мүмкіндік береді.
