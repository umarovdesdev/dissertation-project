## 2. Слайд мазмұны

![Жүйенің жалпы архитектурасы](../assets/architecture/06_system/02_system_architecture.png)

**Біртұтас модель: model = preprocessing + CNN**

| # | Блок | Қызметі / шығысы |
|---|------|-------------------|
| 1 | **Кіріс** | Fundus суреті (BGR uint8) → сол/оң көзге бөлу (s_L, s_R) |
| 2 | **Препроцессинг 𝒫** | 8 кезең: инференсте детерминистік, оқытуда стохастикалық (5, 6 кезеңдер); шығыс — тензор (4, 512, 512): RGB + FOV mask |
| 3 | **CNN backbone** | ResNet-50 немесе EfficientNet-B3, 4-каналды conv1; екі көз бойынша **shared weights** → feature maps + GAP, f ∈ ℝᵈ |
| 4 | **Patient-Level Aggregation Φ** | CNN-нен **тыс** оқытылмайтын оператор; max-grade: ŷ_patient = max(ŷ_L, ŷ_R) |
| 5 | **Prediction Layer g** | softmax (K = 5): p̂ ∈ Δᴷ, ŷ = argmax p̂; referral ŷ_ref = 𝟙{ŷ ≥ 2} |
| 6 | **Grad-CAM модулі** | post-hoc: feature maps + ∂z_c/∂A → heatmap Lᶜ (512×512); метрикалар: **ALO** (бастапқы), IoU |

**Орталық идея:** препроцессинг 𝒫 жеке саты емес, ŷ = CNN_θ(𝒫(I, s)) өрнегінің ажырамас компоненті.

## 3. Баяндаушы сөзі

Диаграммада зерттеудің орталық тезисі — техникалық деңгейде көрсетілген. 

Ұсынылған архитектура дизайнының маңызды бөлігі жеке шығарылған shared weights. CNN-нен тыс жеке оператор ретінде шығарылған Patient-Level Aggregation блогы пациент деңгейіндегі шешім қабылдауды айқын етеді.

Соңында, Grad-CAM модулі heatmap құрастырып, ALO метрикасы арқылы зақым маскаларымен сандық бағаланады.

---

## 4. Қосымша — Терминдер

- **FOV mask (Field of View mask)** — fundus суреттегі көру өрісінің шеңберлі облысын белгілейтін бинарлы маска; 4-ші канал ретінде CNN-ге беріледі.
- **Shared weights** — екі көз (left / right) үшін бір CNN-нің бірдей параметрлерінің қолданылуы; пациент деңгейінде параметр санын екі есе арттырмайды.
- **Patient-Level Aggregation Φ** — сол/оң көздердің болжамдарын біріктіретін оқытылмайтын оператор; CNN-нен бөлек, max-grade стратегиясы — ең ауыр сатыны таңдау.
- **Softmax** — логит-векторды (K = 5) ықтималдық таралымына түрлендіру: p̂ ∈ Δᴷ, Σp̂ᵢ = 1.
- **Referral threshold (ŷ_ref)** — клиникалық бағыттау шешімі: ŷ ≥ 2 болғанда офтальмологқа жіберу (moderate DR немесе одан жоғары).
- **Grad-CAM (Gradient-weighted Class Activation Mapping)** — класс логитінің CNN feature map-тарына қатысты градиенті арқылы есептелетін визуалды түсіндірмелі heatmap.
- **ALO (Attention–Lesion Overlap)** — Grad-CAM активациясының зақым маскаларымен қабаттасу үлесі; H-5 гипотезасының бастапқы метрикасы.
- **GAP (Global Average Pooling)** — feature map-тардың кеңістіктік орташасын алып, оларды векторға айналдыратын қабат.
