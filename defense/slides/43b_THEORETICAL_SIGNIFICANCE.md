## 2. Слайд мазмұны

**1. model = preprocessing + CNN формализациясы**
Алдын ала өңдеу процесі деректерді дайындау сатысы емес, классификациялау моделімен интеграцияланған біртұтас архитектураның ажырамас компоненті ретінде формализацияланды.

**2. ALO (Attention–Lesion Overlap) метрикасы**
Grad-CAM активациясының зақым маскаларымен қабаттасу үлесін зақым типі бойынша есептейтін, CNN-нің клиникалық түсіндірілгіштігін сандық бағалаудың жаңа әдіснамалық негізі.

**3. Препроцессинг арқылы domain-shift төзімділігі**
Архитектурадан тәуелсіз cross-dataset жалпылауды препроцессинг деңгейінде қамтамасыз етудің теориялық дәйектемесі (G ≥ 0.85 критерийімен расталды).

---

## 3. Баяндаушы сөзі

Зерттеудің теориялық маңыздылығы үш негізгі үлеспен сипатталады. Біріншіден, алдын ала өңдеу процесі деректерді дайындау сатысы ретінде емес, CNN-нің ажырамас архитектуралық компоненті ретінде формализацияланды — model = preprocessing + CNN. Екіншіден, ALO метрикасы CNN-нің клиникалық түсіндірілгіштігін сандық бағалаудың жаңа әдіснамалық негізін қалыптастырады; бұған дейін Grad-CAM тек сапалық визуализация ретінде ғана қолданылған. Үшіншіден, препроцессинг арқылы domain-shift-ке төзімділікті архитектурадан тәуелсіз қамтамасыз етуге болатыны теориялық тұрғыдан негізделді және 8 деректер жиынында эксперименттік расталды.

## 4. Қосымша — Терминдер

- **Domain shift** — оқыту мен тест датасеттерінің бөлулері арасындағы айырмашылық (камера, протокол, популяция); модельдің сыртқы датасеттерде өнімділігін төмендететін негізгі себеп.
- **G (generalization ratio)** — F1_external / F1_EyePACS қатынасы; H-4 шегі ≥ 0.85.
- **ALO (Attention–Lesion Overlap)** — Grad-CAM активациясының зақым маскаларымен қабаттасу үлесі (формуласы: ALO = |GradCAM ∩ lesion| / |lesion|).
