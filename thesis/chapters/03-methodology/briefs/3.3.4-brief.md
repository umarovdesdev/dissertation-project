### CONTENT SPECIFICATION

**Section objective:** Specify the training loss — Focal Loss with γ=2 and α set to inverse-frequency class weights (RESEARCH_ARCHITECTURE §4.0; TAB-3.1) — as the principal lever against EyePACS class imbalance for the five-class ordinal DR task, state its formula, relate the inverse-frequency weighting to the class distribution (§4.1.2) and the down-weighting of easy examples to the minority-class problem, and flag the focal-loss primary-source corpus gap without fabricating a citation.

**Argumentative spine (required):**
- **Thesis of this section:** The network is trained with an inverse-frequency-weighted focal loss (γ=2), chosen because the severe, ordinal class imbalance of EyePACS makes unweighted cross-entropy dominated by the majority (DR 0) class; the focal modulation down-weights easy, well-classified examples and the inverse-frequency α up-weights rare severe grades, making this loss the principal of the two imbalance levers (with augmentation).
- **Reasoning chain:**
  1. Problem: EyePACS is severely imbalanced (SB-2.1) and the grades are ordinal (DR 0–4); unweighted cross-entropy would be dominated by DR 0.
  2. Focal loss: FL = −α_t (1 − p_t)^γ log p_t, γ=2 down-weights easy examples so training focuses on hard/minority cases (RESEARCH_ARCHITECTURE §4.0; FIG-3.12).
  3. α = inverse-frequency class weights: rarer grades weighted up in proportion to their scarcity (ties to §4.1.2 class distribution).
  4. Two levers (SC-1.4): the loss (principal) + augmentation (§3.1.3); cost-sensitive learning is an established but under-validated direction (#40: only 2/173 validation studies; SIR-3).
  5. Ordinal connection: the ordinal misclassification cost is also reflected in the quadratic-weighted Cohen's κ used for evaluation (EH-1) — the loss addresses imbalance at training, κ penalizes ordinal error at evaluation (→ §3.4.1).
  6. Corpus gap: the focal-loss canonical primary source (Lin et al. 2017) is not in the assembled corpus (flagged from §2.2.2); specify the loss per governance, flag the citation gap, do not fabricate.
- **Conclusion / hand-off:** With adaptation, initialization, schedule, and loss specified, §3.3 is complete; §3.4 specifies the evaluation framework (metrics, CV, statistics).
