# GLOSSARY_KZ

## EN → KZ Translation Control Glossary

### Doctoral Dissertation: Automated Diabetic Retinopathy Diagnosis via Fundus Image Enhancement and CNN Classification

------------------------------------------------------------------------

## TRANSLATION POLICY

This glossary is designed for controlled translation from English to
Kazakh within the context of the doctoral dissertation.

Principles:

1.  Core technical model names, architectures, metrics, algorithms, and
    abbreviations remain in English.
2.  Conceptual terms are translated into academically appropriate
    Kazakh.
3.  When necessary, first mention may use bilingual form: Kazakh
    (English).
4.  No over-translation of globally standardized AI terminology.
5.  Terminology must remain consistent across chapters, articles, and
    defense materials.

# D. TERMINOLOGY STABILITY RULES

1.  Use "кескін" instead of "бейне".
2.  Use "деректер жиынтығы" instead of "деректер жинағы".
3.  Use "тиімділік" for effectiveness, not "дәлдік" unless referring to
    Accuracy metric.
4.  Accuracy metric remains "Accuracy" unless explanatory context
    requires translation.
5.  Do not translate standardized AI metrics.
6.  Avoid literal translation of globally accepted technical terms.

------------------------------------------------------------------------

# E. STYLE CONTROL FOR ACADEMIC TRANSLATION

• Maintain formal academic Kazakh. • Avoid conversational constructions.
• Preserve logical structure of original English sentence. • Do not
simplify methodological terminology. • Do not localize internationally
standardized algorithm names.
