# What this gives for the dissertation (a very important point)

This table allows us to write the section:

**Dataset acquisition**

example:

> Fundus images were collected from ophthalmology clinics in Kazakhstan and Central Asia using non-mydriatic fundus cameras (Topcon TRC-NW400, Canon CR-2 AF, Zeiss Clarus 500).

Why this is good:

* the same cameras are used in **APTOS / EyePACS**
* therefore **domain shift is minimal**
* this increases the **transferability of the model**
